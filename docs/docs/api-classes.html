Classes

This is an example of how html is rendered. you can cgange any class you want by below options. 


<div class="doZoom doZoom-hover doZoom-moving doZoom-ready" 
		bgsrc="background.jpg">  
		Any Content  
</div>
Option Type Default Description
loadingClass String doZoom-loading Class during loading image.
readyClass String doZoom-ready Class after load image and ready to dozoom.
hoverClass String doZoom-hover' Class when mouse enter background container.
leaveClass String doZoom-leave Class when mouse leave background container.
movingClass String doZoom-moving Class when mouse movineg over background container.
errorClass String doZoom-error Class during error happend and image not loaded.
notifyClass String doZoom-notify Class for Error notification.
